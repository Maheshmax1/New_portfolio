# Mahesh Mariappan - Fullstack Developer Portfolio

Welcome to my personal portfolio website! This project is a highly responsive, multi-themed developer showcase designed to outline my journey, skills, and projects.

---

## 🎨 Kept Themes

This portfolio features exactly 5 completely different visual layouts that visitors can swap dynamically using the floating draggable icons at the bottom-right of the screen:
1. **Apple Style**: Minimal, high-contrast, card-based premium layout.
2. **Minimalist**: Plain black & white text-focused aesthetic with deep margins.
3. **Creative Art**: Bold borders, neon contrasts, and neobrutalist elements.
4. **Dev Console**: A terminal-style CLI dashboard with monospace font sheets.
5. **Dark Neon**: Dark theme featuring glowing purple visual gradients.

---

## 💻 Tech Stack & Integrations

- **Core Framework**: Next.js 15 (Static HTML Export `output: 'export'`)
- **UI Logic**: React 19, TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Form Integration**: Direct WhatsApp submissions redirecting to `https://wa.me/916379095634`

## 🚀 How to Run Locally

1. **Install dependencies**:
   ```bash
   npm install
   ```
2. **Start the development server**:
   ```bash
   npm run dev
   ```
3. **Build the static site**:
   ```bash
   npm run build
   ```
   *The static assets will output to the `/out` directory, which can be deployed instantly to GitHub Pages or Vercel.*
